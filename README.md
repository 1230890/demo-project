# Demo Project

一个简洁风格的个人欢迎页面，包含弹窗按钮、计数器和待办清单三个互动模块。

## 目录结构

```
demo-project/
├── index.html          # 主页面
├── README.md
├── .gitignore
└── assets/
    ├── css/
    │   └── style.css   # 全局样式
    ├── js/
    │   ├── script.js   # 弹窗按钮
    │   ├── count.js    # 计数器（-1 / +1）
    │   └── todo.js     # 待办清单（增删 + 回车快捷添加）
    ├── images/         # 图片资源
    └── lib/            # 第三方库（Bootstrap 等）
```

## 功能模块

| 模块 | 文件 | 说明 |
|------|------|------|
| 弹窗按钮 | `script.js` | 点击弹出欢迎语 |
| 计数器 | `count.js` | 支持 -1 / +1，最小值为 0 |
| 待办清单 | `todo.js` | 新增、删除、回车快捷添加、空状态提示 |

## 使用方式

直接在浏览器中打开 `index.html` 即可预览，无需构建工具。

## 技术特点

- **纯原生实现**：HTML + CSS + JavaScript，无框架依赖
- **ES6 语法**：箭头函数、`const` / `let`、模板字面量
- **响应式适配**：支持平板（768px）、手机（480px）、超小屏（380px）
- **模块化拆分**：每个功能独立 JS 文件，职责清晰
